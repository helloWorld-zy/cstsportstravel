## 四、测试用例

### 登录流程

```
1. 打开小程序 → 展示登录页
2. 点击"抖音一键登录" → 调用 tt.login → 获取 code
3. 首次登录 → 提示绑定手机号
4. 输入手机号 + 验证码 → 绑定成功 → 跳转首页
5. 非首次登录 → 直接跳转首页
6. 手机号登录 → 输入手机号 + 验证码 → 登录成功
```

### 产品浏览流程

```
1. 首页 → 点击产品列表 → 展示产品列表
2. 筛选 → 选择目的地/天数/价格 → 列表更新
3. 点击产品 → 进入详情页 → 展示产品信息
4. 查看行程 → 展示每日行程安排
5. 查看费用说明 → 展示费用包含/不包含
6. 查看退改规则 → 展示退改政策
```

### 预订支付流程

```
1. 产品详情 → 点击"立即预订" → 进入预订页
2. 选择团期 → 选择日期和人数
3. 填写出游人 → 姓名+手机号+身份证
4. 确认订单 → 展示订单详情和价格
5. 选择支付方式 → 抖音支付
6. 调起收银台 → 完成支付
7. 支付成功 → 跳转订单详情
```

### 订单管理流程

```
1. 个人中心 → 我的订单 → 订单列表
2. 切换 Tab → 全部/待付款/已付款/已完成/已取消
3. 点击订单 → 订单详情
4. 待付款订单 → 去支付
5. 已付款订单 → 申请退款
```
